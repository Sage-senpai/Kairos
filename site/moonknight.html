Move-native agent framework

An AI agent that moves real money on Sui.

KAIROS wires a language model to a Sui wallet. Type a sentence and it builds the programmable transaction, signs it, sends it, and hands back the digest. Swaps on DeepBook, memory on Walrus, any object on chain.

01

One transaction, or none

Transfer, swap, stake, read. A swap and the transfer that follows it ride in a single programmable transaction. It all lands or nothing does, so there is no half-finished state to babysit.

02

Memory it keeps on Walrus

The agent stores what it does on Walrus, not in a database you rent. Kill the process, boot it back next week, and it still remembers. The user owns those blobs.

03

Actions in about 20 lines

An action is a TypeScript object with a plain-English description. The model reads it and decides when to call. You write no router, no parser, no state machine.

04

Agents that coordinate

One agent writes a summary to Walrus, another reads it back as context. They sync through the chain instead of calling each other, so nothing leaks between them.

05

Built on Claude tool use

Every action becomes a tool the model can call. It picks, you execute, the result feeds back into the conversation. That loop is the whole framework.

Three commands to a live agent

Scaffold it, drop in two keys, boot it. Then talk to it over HTTP.

# scaffold a new agent
npx create-kairos-agent Atlas

# add SUI_PRIVATE_KEY and ANTHROPIC_API_KEY, then boot
cd agents/atlas && cp .env.example .env
pnpm install && pnpm start
