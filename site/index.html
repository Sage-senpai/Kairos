Agents that act on Sui

KAIROS is a Move-native AI agent framework for the Sui blockchain. Give an LLM a Sui wallet, a personality, and a set of actions — it signs PTBs, swaps on DeepBook, stores memory on Walrus, and reads any Sui object, all from natural language.

Get started How it works

Native Sui actions

Transfer, swap, stake, and read objects — PTBs composed and signed atomically.

User-owned memory

Agent memory lives on Walrus and survives restarts. The user owns it, not a database.

20-line actions

Add a capability in a few lines of TypeScript. The LLM reads the description and calls it.

Multi-agent by design

Agents coordinate through Walrus and Sui objects — no direct calls, no shared secrets.

Boot an agent

# 1. Scaffold
npx create-kairos-agent Atlas

# 2. Configure and run
cd agents/atlas
cp .env.example .env   # fill in your keys
pnpm install && pnpm start

# 3. Talk to it
curl -X POST http://localhost:3000/message \
  -H "Content-Type: application/json" \
  -d '{"text": "What is my SUI balance?", "userId": "alice"}'
